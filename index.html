Welcome to Managed Commons.

Managed Commons is the open source project for a set of fully managed libraries for Mono/.NET.

Already in the repository we have 3 libraries:

  • Commons.GetOptions - Attribute-directed parsing of command line parameters. Evolution of old Mono.GetOptions
  • Commons.Locale.Helpers.pt_BR - Textual helpers for the Brazilian Portuguese language. Main feature: a flexible number-to-text converter.
  • Commons.WebM - Decoding of WebM video streams

Managed-Commons maintained by monoman
